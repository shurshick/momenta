# Changelog

## [v0.2.77] - 2026-07-16

### Fixed

- **Android**: на экране настроек исправлено авторство приложения на Александра Коваленко.

### Changed

- **Android**: из настроек убраны API URL, адрес медиа-сервера и внутренний flavor сборки.
- **Android**: удалены неиспользуемые строки старой диагностики соединения.
- **Release**: серверных и Room-миграций нет; Alembic head `008`, Room schema `7`.

## [v0.2.76] - 2026-07-15

### Fixed

- **Posts**: PostgreSQL advisory lock не позволяет параллельным запросам превысить дневной лимит пользователя.
- **Worker**: несколько процессов больше не могут одновременно обрабатывать один пост.

### Added

- **Worker**: DB leases с владельцем, таймаутом и автоматическим восстановлением зависших задач.
- **Migration**: Alembic `008` добавляет lease-поля и индекс очереди обработки.
- **Migration**: добавлена отсутствовавшая базовая ревизия `000`, поэтому чистая установка теперь действительно доходит до head.
- **CI**: полный backend-прогон на PostgreSQL 16, реальные Redis/MinIO проверки, Docker `/ready` smoke test и dump/restore drill.

### Changed

- **Worker**: `WORKER_CONCURRENCY` теперь реально задаёт число параллельных media-задач; блокирующие S3/Pillow операции вынесены из event loop.
- **Dependencies**: удалена неиспользуемая зависимость RQ; очередь остаётся транзакционной и хранится в PostgreSQL.
- **Release**: server-only, Android остаётся на `0.2.73`; Alembic head `008`, Room schema `7`.

### Tests

- Добавлены регрессии конкурентной публикации, повторного claim и интеграции с внешним стеком.
