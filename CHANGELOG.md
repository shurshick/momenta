# Changelog

Все заметные изменения проекта Momenta / Момент описаны здесь.

## [v0.2.30] - 2026-06-28

### Changed
- **Android**: выполнен общий UI-polish основных экранов: "Момент дня", лента, профиль, камера и публикация.
- **Android**: карточка задания дня стала ниже и плотнее, лучший момент дня визуально читается лучше.
- **Android**: лента получила аккуратный skeleton-loading, более стабильные карточки постов и единые поля ввода комментариев.
- **Android**: профиль стал плотнее: интерактивный аватар с edit-индикатором, компактная статистика и более заметный блок недавних моментов.
- **Android**: камера получила более читаемую нижнюю панель, оформленные кнопки "Галерея" и "Эффекты", компактный выбор эффектов.
- **Android**: экран публикации стал ниже по вертикали, чтобы поле подписи и кнопка отправки ощущались устойчивее при клавиатуре.

## [v0.2.29] - 2026-06-28

### Changed
- **Android**: карточка задания дня стала еще компактнее по высоте.
- **Android**: в карточке лучшего момента дня убрана нижняя кнопка "Смотреть мир сейчас"; переход в ленту остается по тапу на карточку.
- **Android**: блок "Недавние моменты" в профиле стал статичной сеткой 3x3 без внутренней прокрутки.
- **Docs**: восстановлена UTF-8 кодировка в README, changelog, API и TrueNAS-документации.

## [v0.2.28] - 2026-06-28

### Changed
- **Android**: карточка задания дня стала компактнее, чтобы лучший момент дня помещался выше на экране.
- **Android**: статистика профиля перенесена справа от аватара, блок недавних моментов поднят выше.

### Fixed
- **Android**: русское склонение счетчика дней подряд теперь корректное: день, дня, дней.
- **Backend**: версия в админке и `/api/v1/meta` больше не зависит от устаревшего `APP_VERSION` в окружении.

## [v0.2.27] - 2026-06-28

### Fixed
- **Android**: после публикации нового момента лента обновляется последовательно и повторно проверяет сервер, поэтому свежий пост появляется без переключения вкладок.

## [v0.2.26] - 2026-06-28

### Added
- **Backend**: `/api/v1/challenges/today` автоматически создает задание дня, если активного задания на дату нет.
- **Backend**: добавлены поля `prompt` и `source`, локальные шаблоны автозаданий и настройка `APP_TIMEZONE`.
- **Android**: набор аватарок заменен на 40 новых портретных PNG из утвержденного набора.

### Changed
- **Android**: главный экран показывает задание дня и лучший момент дня двумя независимыми блоками.
- **Android**: после публикации лента скроллится наверх только один раз; повторные refresh больше не кидают пользователя к началу, если он уже листает.

### Fixed
- **Android**: DTO задания дня теперь совместим с фактическим ответом backend.
- **Backend**: автогенерация задания дня идемпотентна и не падает при конфликте уникальности.

## [v0.2.25] - 2026-06-27

### Fixed
- **Android**: исправлена бесконечная загрузка профиля на первом открытии.
- **Android**: лучший момент дня загружается независимо от карточки задания дня.

## [v0.2.24] - 2026-06-27

### Changed
- **Android**: профиль обновляется тихо при возврате на экран, без полного лоадера и заметного моргания.
- **Android**: главный экран повторно обновляет лучший момент дня при возврате на вкладку.
- **Android**: кнопка скачивания APK открывает системный установщик.
- **Release**: APK в GitHub Actions подписывается стабильным update-keystore.
- **Backend**: лучший момент дня использует fallback по активным постам за последние 48 часов.

## [v0.2.23] - 2026-06-27

### Added
- **Backend**: добавлен `/api/v1/users/suggestions` для верхнего ряда реальных пользователей в ленте.
- **Android**: набор предопределенных аватарок расширен и заменен PNG-портретами.

### Changed
- **Android**: верхний ряд ленты показывает реальные имена и аватарки пользователей вместо заглушек.
- **Android**: после публикации запускается retry-refresh ленты.
- **Android**: главный экран добирает момент дня из свежей ленты, если endpoint временно вернул пусто.
- **Android**: профиль обновляет счетчики при возврате на экран.

## [v0.2.22] - 2026-06-27

### Added
- **Android**: в окне "О программе" появилась кнопка скачивания APK при найденном обновлении.
- **Backend**: добавлены тесты для `/api/v1/feed/today/best-random` и первой страницы ленты.

### Changed
- **Android**: после публикации приложение открывает свежую ленту.
- **Android**: улучшен UI выбора эффектов камеры.
- **Android**: обработка фото с эффектами нормализует EXIF-ориентацию.
- **Backend**: worker добавляет активированные посты в Redis feed zset.

## [v0.2.21] - 2026-06-27

### Fixed
- **Android**: имя установленного приложения изменено на "Момент".
- **Android**: исправлен refresh token flow для 401.
- **Android**: убраны фоновые дублирующиеся загрузки ленты при обычном открытии экрана.

## [v0.2.20] - 2026-06-27

### Changed
- **Android**: исправлены подписи "Момент дня" и "Момент".
- **Android**: выбор аватара открывается по тапу на аватар.
- **Backend**: лучший момент дня выбирает случайный пост из топа за день, с fallback для тестовой базы.

## [v0.2.19] - 2026-06-27

### Added
- **Android**: экран "О программе" с проверкой обновления.
- **Android**: новая иконка приложения.
- **Android**: статичные пейзажи на onboarding.

## [v0.2.18] - 2026-06-27

### Added
- Pull-to-refresh ленты.
- Комментарии.
- `avatar_key`.
- Удаление своих постов в течение 24 часов.
- Лучший момент дня из топа.

### Fixed
- Логика лайков.
- Применение эффектов камеры.

## [v0.2.0 - v0.2.17] - 2026-06-25..2026-06-27

### Summary
- Первый рабочий Android APK с интеграцией backend.
- Auth, camera, publish, feed, profile, settings.
- Админка, Docker/TrueNAS deploy, MinIO/S3, worker, CI.
- Исправления установки APK, `android:testOnly`, Jinja2 admin templates, SQLite-совместимости тестов и базовой интеграции Android/backend.

## [v0.1.0] - 2026-06-25

### Added
- Базовый FastAPI backend.
- Регистрация, вход, refresh tokens.
- Посты, реакции, жалобы, лента.
- Профиль пользователя.
- Admin panel.
- PostgreSQL, Redis, MinIO/S3, Docker Compose.
