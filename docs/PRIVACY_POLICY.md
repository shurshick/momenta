# Политика конфиденциальности «Момент»

Дата обновления: 16 июля 2026 года.

## Какие данные использует приложение

- Данные аккаунта: имя пользователя, email, отображаемое имя, описание и выбранный аватар.
- Публикации: фотографии, подписи, дата задания и необязательные страна/город.
- Действия в сервисе: реакции, комментарии, избранное и жалобы.
- Технические данные запросов, необходимые для авторизации, защиты сервиса и диагностики ошибок.

## Для чего нужны данные

Данные используются только для работы аккаунта, публикации моментов, формирования ленты, взаимодействия пользователей, модерации и защиты сервиса.

## Где хранятся данные

Данные аккаунтов и публикаций хранятся на серверной инфраструктуре «Момент». На устройстве приложение временно хранит токены сессии, кэш ленты и файлы ожидающих отправки публикаций. Кэш можно очистить в настройках; файлы очереди при этом сохраняются.

## Передача данных

Приложение не продаёт персональные данные. Фотографии, профиль и действия, предназначенные для публикации, видны другим пользователям сервиса. APK обновлений загружается из официальных GitHub Releases проекта.

## Управление данными

Профиль можно изменить в приложении. Для вопросов о данных и удаления аккаунта используйте раздел «Сообщить о проблеме» в настройках. Не публикуйте в обращении пароль или токены доступа.
